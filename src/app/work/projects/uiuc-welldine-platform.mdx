# UIUC WellDine: Comprehensive Food Delivery Platform

A full-stack food delivery and takeout platform built for UIUC, providing comprehensive food ordering, delivery management, and restaurant services through modern web technologies and scalable backend architecture.

## Overview

WellDine is a comprehensive food delivery platform designed to streamline food ordering and delivery services for university students and staff. The platform provides restaurant browsing, order management, delivery tracking, and administrative tools through a user-friendly web interface.

## Key Features

- **User Management**: Comprehensive user profiles and role-based access
- **Food Ordering**: Restaurant browsing, menu management, and order placement
- **API Integration**: Extensive RESTful API ecosystem
- **Performance Optimization**: High-speed response times and system efficiency
- **Scalable Architecture**: Built to support growing user base

## Technical Implementation

### Backend Architecture
- **Framework**: Java Spring Boot for robust enterprise-level development
- **Database**: Redis for high-performance caching and session management
- **Data Processing**: MyBatis for efficient database operations
- **API Design**: RESTful architecture with comprehensive endpoint coverage
- **Frontend**: JavaScript, HTML5, Nginx for responsive user interfaces

### Development Approach
- **API-First Design**: 65 RESTful APIs supporting diverse development tasks
- **Performance Focus**: Strategic optimization for enterprise operations
- **User Experience**: Intuitive interfaces for food ordering and delivery management
- **Enterprise Integration**: Seamless integration with existing university systems
